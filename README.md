# compiler
Compilador desenvolvido para a matéria Lógica Computacional Entrega 1

## Rodando
Para testar a calculadora:  
Exemplo 1:  
```$ python3 main.py "2+ 5 -3"```  
Exemplo 2:  
```$ python3 main.py "12-32+100"```  
Exemplo 3  
```$ python3 main.py "58   + 10  - 2"```
